# Personalization
## Keyboards shortcuts
Settings > Keyboard > View and Customize Shortcuts

### Launchers
Home Folder = Super + E
Launch web browser = Super + W
Search = Alt + Space
Settings = Super + I

### Navigation
Switch Applications = Super + Tab
Switch windows = Alt + Tab

### System
Show the overview = Disable

### Windows
Activate the windows menu = Disable
