# my-debian

Basically my script to install debian as easy as possible.

## Download Debian

https://brockar.github.io/easy-download-debian/

# How to use

Run as su (in debian) or sudo.

```
sudo apt install git
```

```
git clone https://github.com/brockar/my-debian.git
cd my-debian/
sudo ./script.sh
```
